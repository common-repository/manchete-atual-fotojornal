Fotojornal, Galeria de Notícias em Imagem, personalizavel, disponibilizado com conteúdos do site Manchete Atual.

== Description ==

Agora pode ter um Fotojornal no seu site ou blog com as notícias mais importantes do dia ilustradas com fotos selecionadas pelo Manchete Atual.
O Fotojornal usa design responsivo para se adaptar ao seu site e pode ser personalizado nas cores e temas das notícias.

Nota:
Este plugin efectua pedidos de dados remotos ao nosso servidor do Manchete Atual.
Um pedido inicial, carrega o conteúdo mínimo para mostrar o fotojornal.
Todos os restantes pedidos são gerados pelos utilizadores ao navegarem no fotojornal para verem outras notícias.

== Installation ==

1. Descarregue o plugin no menu plugins do WordPress ou coloque o conteúdo do arquivo ZIP na pasta `/wp-content/plugins/`.
2. Ative o plugin no menu plugins do WordPress
3. Configure o widget com valores à sua escolha: título, cores, fonte e categoria a mostrar.

== Frequently asked questions ==

= Que categorias posso mostrar em meu site com o player Fotojornal? =
Estão disponíveis as seguintes categorias: brasil; ciencia_saude; economia; esporte; mundo; tecnologia; e variedades.

= Posso combinar categorias? =
Não. No entanto, caso não escolha nenhuma categoria, todas serão mostradas.

= Gostava de usar o Plug-In num site que não tem WordPress. É possível? =
Sim. Mais informações sobre como o fazer em [Manchete Atual Fotojornal](http://mancheteatual.com.br/fotojornal)
